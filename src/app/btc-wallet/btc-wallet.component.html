Bitcoin Wallet

This project features a fictional Bitcoin wallet interface to buy and sell bitcoins, showing the correspondent amount in United States Dollars (USD). The exchange rate is provided by a public API used as a service in Angular. After the operation, the balance is updated and don't reset while navigating on this website.
